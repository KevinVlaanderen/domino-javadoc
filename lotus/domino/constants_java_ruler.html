Java constants: ruler

Where used:

  • FirstLineLeftMargin in RichTextParagraphStyle
  • LeftMargin in RichTextParagraphStyle
  • RightMargin in RichTextParagraphStyle
  • setTab in RichTextParagraphStyle
  • setTabs in RichTextParagraphStyle
  • Position in RichTextTab

Constants:

  • RichTextParagraphStyle.RULER_ONE_CENTIMETER (567)
  • RichTextParagraphStyle.RULER_ONE_INCH (1440)

The unit of measure used by the above constants is a "twip," where one centimeter is 567 twips and one inch is 1440 twips. You can apply arithmetic to the constants to obtain other measurements. For example, RULER_ONE_CENTIMETER * 2 for two centimeters or RULER_ONE_INCH * 0.75 for three-quarters of an inch.
